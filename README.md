# PrimeNumber
Данный репозиторий посвящен библеотеке для работы с огромными простыми числами.
Внутри репозитория вы сможете найти:
- исходный код библиотеки
- `MakeFile` для сборки библиотеки
- программу для тестирования библиотеки
- файл с подробным описанием сборки, тестииования и использования библиотеки
